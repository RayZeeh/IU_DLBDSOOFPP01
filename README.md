# Studiums Dashbaord Demo

## Voraussetzungen

- Python >=3.10.0

## Installation

1. Repository klonen:

   ```bash
   git clone https://github.com/RayZeeh/IU_DLBDSOOFPP01
   cd IU_DLBDSOOFPP01
   ```

2. Virtuelle Umgebung erstellen und aktivieren:

    ```bash
   # Windows
   python -m venv venv
   .\venv\Scripts\activate
   ```

   ```bash
   # Linux/Mac
   python -m venv venv
   source venv/bin/activate
   ```
